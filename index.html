Sobre mí

Soy profesional en negocios internacionales, con enfoque en desarrollo de software. Me apasiona la tecnología, en especial el desarrollo de software. He estado aprendiendo de manera autodidacta, especializandome con algunos cursos y practicando con proyectos de aplicativos web.

Mi enfoque se centra en el desarrollo Frontend utilizando tecnologías como HTML, CSS, JavaScript y Bootstrap, complementado con sólidas habilidades en el desarrollo Backend utilizando Java, Spring y MySQL.

Tecnologías

Frontend

HTML: Lenguaje de marcado utilizado para estructurar el contenido de una página web.

CSS: Lenguaje de estilo utilizado para diseñar y dar formato al contenido HTML.

JavaScript: Lenguaje de programación que permite la interactividad en el lado del cliente en una página web.

Bootstrap: Marco de diseño de código abierto que facilita el diseño y la estilización de páginas web responsivas.

Angular: Framework de desarrollo web, utilizado para crear aplicaciones de una sola página (SPA) con una arquitectura robusta y modular.

Backend

Java: Lenguaje de programación versátil utilizado para el desarrollo de aplicaciones, incluyendo el backend.

MySQL: Sistema de gestión de bases de datos relacional.

Spring Boot: Framework que facilita la creación rápida de aplicaciones Java, con configuración mínima, auto-configuración y servidores embebidos para un despliegue sencillo.

Control de versiones

Git: Sistema de control de versiones que permite el seguimiento de cambios en el código fuente.

GitHub: Plataforma basada en web que utiliza Git para el control de versiones y colaboración en proyectos.
